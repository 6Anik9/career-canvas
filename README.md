# 🎨 CareerCanvas | Minimalist CV Generator

**CareerCanvas** is a sleek, minimalist web-based CV builder designed for professionals who value clean typography and high-end design. Create, preview, and export your professional resume in seconds without the clutter of traditional word processors.

---

## ✨ Features

* **Live Preview:** See changes in real-time as you type your details.
* **Minimalist Aesthetic:** Focused on high-end typography and a "canvas" feel.
* **Portrait Support:** Easily upload and preview your professional headshot.
* **Skill Visualizer:** Dynamic progress bars based on your proficiency levels.
* **Multiple Export Options:**
    * **High-Res PNG:** Powered by `html2canvas`.
    * **Print/PDF:** Optimized print styles for clean paper or digital PDF export.
* **Responsive Design:** Fully functional on mobile and desktop devices.

---

## 🛠️ Built With

* **HTML5** - For the application structure.
* **CSS3** - Custom styling with CSS Variables and `@media` print queries.
* **JavaScript (Vanilla)** - For DOM manipulation and real-time syncing.
* **[html2canvas](https://html2canvas.hertzen.com/)** - To convert HTML sections into downloadable images.
* **[FontAwesome](https://fontawesome.com/)** - For professional iconography.
* **[Google Fonts](https://fonts.google.com/)** - Using the "Inter" and "Georgia" typefaces.

---

## 📸 Usage

1.  **Compose:** Fill in your name, contact info, and professional history in the 'Compose' tab.
2.  **Skills:** Enter skills in the format `SkillName:Level` (e.g., `Design:90, Coding:85`).
3.  **Preview:** Switch to the 'Preview' tab to see your final layout.
4.  **Export:** * Click the **Camera Icon** to download a high-quality `.png` image.
    * Click the **File Icon** to open the print dialog and save as a `.pdf`.
